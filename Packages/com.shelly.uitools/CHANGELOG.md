# Changelog

本檔記錄各版本變更，版本號遵循 [語意化版本 SemVer](https://semver.org/lang/zh-TW/)。

## [1.2.0] - 2026-06-18

### Added
- `CanvasResolutionHandler` — Canvas 解析度自適應元件
- `UIVisibility` — CanvasGroup 即時顯示/隱藏 extension methods
- `UIAnimator` — CanvasGroup 動畫顯示/隱藏（fade + scale）
- `UITween` — Scale / Alpha 常用 Tween extension methods


## [1.1.0] - 2026-06-17

### Added
- 新增 PrimeTween 依賴（透過 OpenUPM）

### Changed
- 編輯器選單名由 Yi Hsuan 改為 Shelly


## [1.0.0] - 2026-06-17

### Added
- 初始版本骨架
- 範例 Runtime 元件 `UIToolExample`
- 範例編輯器視窗 `UIToolWindow`
